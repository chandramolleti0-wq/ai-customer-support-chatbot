# AI Customer Support Chatbot

An AI-powered customer support chatbot built using Flask and Google Gemini API.

## Features
- Real-time AI responses
- Modern chatbot UI
- Flask backend
- Gemini AI integration

## Technologies Used
- Python
- Flask
- HTML/CSS/JavaScript
- Google Gemini API

```bash
pip install -r requirements.txt
python app.py
